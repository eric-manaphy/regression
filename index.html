How to use this tool

  • Wait for the tool to load. It takes a little bit when you first load the site and when you first select an option, but subsequent regressions should be instantaneous as long as you do not leave the site. If it doesn't work the first time, click it again. If it doesn't work after that, refresh and then try again.
  • Note: FireFox keeps the state of some HTML elements on refresh, so the buttons may not look disabled on refresh. Do not click the buttons right away. Either go to the site again by hitting Enter in the URL bar or just wait ~30 seconds.
  • Enter TSV data into the text area. The tool assumes the first row is headers,and does not (currently) do any data validation. Make sure that your data is formatted properly and choose the regression model that matches the input.
  • Data should be columns of A, B, C, Vmax in that order. Of course, you only need as many columns as the regression model requires. The main point is that Vmax should be the last column.
