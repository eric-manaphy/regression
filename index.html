How to use this tool

  • Enter TSV data into the left text area. The tool assumes the first row is headers, and does not (currently) do any data validation. Make sure that your data is formatted properly and click the "Validate" button on the right side. Applicable regression models should appear.
  • Once you click on a model, it takes a little bit when you first select an option, but subsequent regressions should be instantaneous as long as you do not leave the site. If it doesn't work the first time, click it again. If it doesn't work after that, refresh and then try again.
  • Data should be columns of A, B, C, Vmax in that order. Of course, you only need as many columns as the regression model requires. The main point is that Vmax should be the last column.
